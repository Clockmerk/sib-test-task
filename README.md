## Purpose of the application
Тестовый проект таблицы напоминаний
<br>
Ссылка на [сайт](https://sib-test-task.vercel.app/)

## Getting Started
```
npm install -g @angular/cli
npm install
ng serve --open
```

## MVP functionality
* Начальный функционал вывода списка
* Базовый роутинг
* Базовый редюсер с localStorage

## Stack
* Angular 17
